Block Level Elements

কোডিং এর ক্ষেত্রে যে সকল জিনিসগুলো নিজে বসার পরে পাশে অন্য কাওকে বসতে দেয় না, সে সকল এলিমেন্টকে Block Level Element বলে।

Inline Elements

যে সকল এলিমেন্ট নিজে বসার পর অন্য একজঙ্কে তার পাশে বসতে দেয় তাকে Inline Element বলে। আবার Block Level Element কে Inline Element এর পাশে একসাথে বসানো যায় না।

Inline Block Element

Inline Element এর মত নিজে বসার পর অন্যকে জায়গা দিবে, কিন্তু নিজের যতটুকু জায়গার প্রয়োজন, তা নিয়ে নিবে। পাশের দিকে Inline, উপরে নিচে Block. মেনুবার, নাভবারে Inline Block Element বেশি ব্যবহৃত হয়।

Semantic Markup

Element এর চরিত্র অনুযায়ী কোন মার্কাপের কি কাজ তা সেই মার্কাপের নাম পরে বুঝতে ও ব্যবহার করতে হবে।
